類語ポップアップ
===
本拡張では、VS Code のエディタで選択した文字列の類語を表示します。

## 使い方
対象のテキストを選択してマウスオーバーする。  
![ss](https://github.com/n-fukuju/synonym-vsce/raw/master/client/images/ss.png)

## 辞書
類語検索の辞書として、日本語ワードネットを使用しています。
